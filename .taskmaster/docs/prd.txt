# 연수김안과의원 AI 기반 디지털 마케팅 랜딩페이지 PRD

## 프로젝트 개요
연수김안과의원을 위한 AI 기반 디지털 마케팅 랜딩페이지 개발 프로젝트

### 프로젝트 목표
- SEO/AEO 최적화로 월 방문자 10,000명+ 달성
- 방문자 → 리드 전환율 35% 이상
- 재방문율 80% 이상, 평균 LTV 150만원 달성
- AI 기반 자동화로 운영 효율성 300% 증대

### 핵심 KPI
- CTR (Click Through Rate): ≥ 6%
- CPL (Cost Per Lead): ≤ ₩20,000
- 예약 완료율: ≥ 35%
- 수술 전환율: ≥ 15%
- 고객 만족도: ≥ 4.8/5.0

## 기술 스택
- Frontend: Next.js 14 (App Router), Tailwind CSS 3.4, TypeScript
- Backend: Vercel Edge Functions, NestJS, Prisma
- Database: PlanetScale (MySQL)
- AI: OpenAI GPT-4o, Gemini, Pinecone 벡터 DB
- CRM 연동: BMX 제품 연동
- 호스팅: Vercel

## 주요 기능

### 1. 랜딩페이지 핵심 섹션
- Hero Section with 30년 신뢰 브랜딩
- AI 챗봇 상담 서비스
- 주요 서비스 (라식/라섹, 백내장 수술, 망막 질환)
- 의료진 소개 (김○○ 원장)
- 병원 정보 및 위치
- 환자 후기 및 수술 실적

### 2. AI 챗봇 시스템
- 증상 분류 (12개 카테고리)
- 검사/수술 매핑
- 실시간 예약 CTA
- 카카오 알림톡 연동
- 24/7 상담 서비스

### 3. 예약 시스템
- 실시간 예약 현황
- 온라인 예약 폼
- 카카오톡 예약 확인
- 리마인드 알림
- 구글 캘린더 연동

### 4. CRM 연동 (BMX)
- 환자 정보 동기화
- 검진 이력 관리
- 재방문 리마인드
- 마케팅 자동화

### 5. SEO/AEO 최적화
- 구조화 데이터 (JSON-LD)
- FAQ 섹션
- Speakable 마크업
- 메타 태그 최적화
- 사이트맵 자동 생성

### 6. 접근성 (KWCAG 2.1 AA)
- 키보드 내비게이션
- 스크린 리더 지원
- 고대비 모드
- 다국어 지원

### 7. 모바일 최적화
- 모바일 퍼스트 디자인
- PWA 지원
- 터치 최적화
- 빠른 로딩 속도

### 8. 마케팅 자동화
- 재검진 리마인드 (6/12/24개월)
- 미예약 이탈자 시퀀스
- 수술 후기 UGC 촉진
- A/B 테스트 시스템

## 페이지 구조
1. 메인 랜딩페이지 (/)
2. 서비스 소개 (/services)
3. 의료진 소개 (/doctors)
4. 예약 페이지 (/booking)
5. 환자 후기 (/reviews)
6. 병원 정보 (/info)
7. AI 상담 (/consultation)

## 법적 준수 사항
- 한국 의료광고심의 규정 준수
- 개인정보보호법 준수
- 정보통신망법 준수
- AI 챗봇 의료 정보 제공 한계 명시

## 성공 지표
- 월 방문자 수: 10,000명+
- 전환율: 35%+
- 재방문율: 80%+
- 페이지 로딩 속도: 3초 이내
- 모바일 사용성 점수: 95점+
- Core Web Vitals: 모든 지표 통과
