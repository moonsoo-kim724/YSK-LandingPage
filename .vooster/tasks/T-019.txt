# FAQ 및 AI FAQ 자동생성 UI 리디자인 및 인터랙션 개선

**Task ID:** T-019
**Status:** BACKLOG
**Importance:** SHOULD
**Complexity:** 6/10
**Urgency:** 7/10
**Dependencies:** T-013, T-016

## Description

### 설명
자주 묻는 질문(FAQ) 페이지와 AI가 자동으로 FAQ를 생성하는 기능(T-013)의 사용자 인터페이스를 새로운 디자인 가이드에 맞춰 개선합니다. 정보 탐색의 편의성을 높이고, 'AI 답변 받기'와 같은 핵심 기능으로 사용자의 참여를 유도하기 위해 인터랙션을 강화합니다.

### 구현 상세
1.  **FAQ 목록 UI 리디자인**: 기존 FAQ 목록을 `shadcn/ui`의 `Accordion` 컴포넌트를 활용하여 새 디자인 시스템에 맞게 재구현합니다. 질문 클릭 시 답변이 부드럽게 펼쳐지는 애니메이션을 적용합니다.
2.  **핵심 CTA 시각적 강조**: 'AI 답변 받기'나 '더 자세한 내용 챗봇에게 물어보기'와 같은 주요 CTA 버튼에 디자인 가이드의 Accent Color(#00A8CC)를 적용하여 사용자의 주목도를 높입니다.
3.  **AI 답변 인터랙션 개선**: AI가 답변을 생성하는 동안(T-013 연동) 스켈레톤 UI나 로딩 인디케이터를 표시하여 시스템이 작동 중임을 명확히 알립니다. 답변이 생성되면 타이핑 효과나 페이드인 효과를 주어 동적으로 표시합니다.
4.  **모바일 최적화**: 모바일 화면에서 FAQ 질문과 답변의 가독성을 높이기 위해 폰트 크기와 줄 간격을 조정합니다. 터치 영역을 충분히 확보하여 사용자가 쉽게 항목을 선택할 수 있도록 합니다.

### 테스트 전략
1.  **인터랙션 기능 테스트**: FAQ 아코디언이 정상적으로 열리고 닫히는지, 모든 질문/답변이 올바르게 표시되는지 확인합니다. 'AI 답변 받기' 버튼 클릭 시 로딩 상태 표시 후 AI 답변이 정상적으로 나타나는지 검증합니다.
2.  **반응형 UI 테스트**: 브라우저 개발자 도구를 사용하여 모바일, 태블릿, 데스크탑 등 다양한 화면 크기에서 FAQ 페이지의 레이아웃이 깨지지 않고 콘텐츠가 잘 읽히는지 확인합니다.
3.  **콘텐츠 로딩 테스트**: AI FAQ 자동 생성 기능(T-013)과 연동하여, 새로 생성된 FAQ가 목록에 올바르게 반영되고 표시되는지 테스트합니다.
4.  **사용성 검증**: 사용자가 원하는 정보를 쉽게 찾을 수 있는지, AI 답변 기능이 직관적으로 이해되고 사용하기 편리한지 동료 리뷰나 간단한 사용성 테스트를 통해 피드백을 받습니다.
